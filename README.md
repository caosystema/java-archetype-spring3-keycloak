# java-archetype-spring3-keycloak
Basic archetype for Java 17/21 with Spring Boot 3 and Keycloak

## Features
- [x] Java 17/21
- [x] Spring Boot 3
- [x] JPA
- [x] Spring Security
- [x] Keycloak as Auth Server
- [x] Clean Architecture
- [x] Exception Controller
- [x] Custom Messages and Exceptions
- [x] OpenAPI/Swagger
- [x] Auditory
- [ ] GraalVM integration
